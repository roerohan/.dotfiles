# Aerospace

i3-like window manager for MacOS.

## Installation

```sh
brew install --cask nikitabobko/tap/aerospace
```

## Config

```sh
mkdir -p ~/.config/aerospace/
ln -s "`pwd`/aerospace.toml" ~/.config/aerospace/aerospace.toml
```

## Configuration Summary

To configure Aerospace, follow these steps:

1. Create the configuration directory:
   ```sh
   mkdir -p ~/.config/aerospace/
   ```

2. Link the `aerospace.toml` configuration file:
   ```sh
   ln -s "`pwd`/aerospace.toml" ~/.config/aerospace/aerospace.toml
   ```

This configuration file will allow you to customize the behavior of the Aerospace window manager according to your preferences.
